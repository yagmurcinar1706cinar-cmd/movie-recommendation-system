# 🎬 Movie Recommendation System
## 👩‍💻 Author

Yağmur Çınar


## 📌 Project Overview

This project implements a content-based recommendation system that suggests similar movies based on their genres.

---

## 🛠️ Technologies Used

* Python
* Pandas
* Scikit-learn

---

## ⚙️ How It Works

* Movie genres are converted into numerical vectors
* Cosine similarity is used to measure similarity between movies
* The system recommends the most similar movies

---

## 🎯 Features

* Recommend movies based on a given title
* Uses content-based filtering
* Simple and efficient approach

---

## 📈 Example

Input:
Narcos

Output:

* Narcos: Mexico
* El Chapo
* Breaking Bad
* Queen of the South
